기본 스케쥴
고정스케쥴 표를 기준으로 매주 같은 패턴을 생성합니다. 수동수정된 칸은 고정스케쥴 변경 시 덮어쓰지 않습니다.
공휴일·주말
공휴일에 원래 근무가 있는 칸은 2시 퇴근으로 자동 반영합니다. 토요일과 일요일도 단축일로 판단합니다.
전체 휴진일
매년 어린이날과 크리스마스는 전체 휴진입니다. 설 연휴 3일과 추석 연휴 3일도 공휴일 이름을 기준으로 전체 휴진 처리하며, 해당 날짜는 최소 인원 검증에서 제외합니다.
일별 최소 진료 인원
기본 기준은 4명입니다. 평일 중 화요일·목요일은 최소 3명 이상이면 정상으로 판단합니다.
퇴근 시간 검증
공휴일이 아닌 평일에는 8시 퇴근 1명, 7시 퇴근 최소 1명이 있어야 정상으로 봅니다.
주간 근무 시간 검증
대표원장을 제외한 원장님은 평일 근무일의 퇴근 시간 평균이 7시가 되도록 확인합니다.
연차 사용
월간 진료표에서 연차사용으로 바꾼 날짜만 근태관리의 사용 내역과 총 사용일수에 반영합니다.
1:1 스왑
근무 중인 칸에서만 신청할 수 있고, 해당 날짜에 휴무인 원장님만 대상으로 선택할 수 있습니다.
저장·이동 보호
현재 데이터 저장과 관리자 설정 진입은 비밀번호가 필요합니다. 저장하지 않은 변경이 있을 때 주 이동을 하면 저장 후 이동, 저장하지 않고 이동, 이동 취소 중 선택합니다.

적용 시작일 이후의 기본생성/공휴일반영 스케쥴만 새 고정스케쥴로 덮어씁니다. 수동수정한 칸은 유지됩니다.
